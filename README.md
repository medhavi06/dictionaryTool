# dict
Command Line Dictionary Tool.

## Setup
1. Install node, npm.
2. Download the repository, unzip the repository.
    ```
    npm link
    ```
3. The possible commands are,
    * *dict defn ```<word>```*
    * *dict syn ```<word>```*
    * *dict ant ```<word>```*
    * *dict ex ```<word>```*
    * *dict ```<word>```*
    * *dict play*
    * *dict*
